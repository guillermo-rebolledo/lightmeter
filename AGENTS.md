## Agent skills

Use `/i-have-adhd` at the start so that your answers are clear and easy to follow.
Use `/swiftui-pro`, `swiftdata-pro`, and `swift-testing-pro` for developing guidance.

### Issue tracker

Issues and PRDs are tracked as GitHub issues in `guillermo-rebolledo/lightmeter` (via the `gh` CLI). See `docs/agents/issue-tracker.md`.

### Triage labels

Default five canonical labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
